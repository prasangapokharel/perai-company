## How to view diagrams

All diagrams are written in **Mermaid**. They render automatically on GitHub, GitLab, and in
VS Code (with the *Markdown Preview Mermaid Support* extension). For the final printed report,
export each diagram as an image via <https://mermaid.live> and paste it into the Word/LaTeX copy.

## Report standard

The chapter structure follows the TU BCA 8th semester project report guideline:

1. Chapter 1 — Introduction
2. Chapter 2 — Background Study and Literature Review *(merged into Chapters 1–2 here)*
3. Chapter 3 — System Analysis
4. Chapter 4 — System Design
5. Chapter 5 — Implementation and Testing
6. Chapter 6 — Conclusion and Future Recommendations
